# Live Markdown Previewer

Full-stack web app that converts Markdown to HTML in real time, built to showcase a complete CI/CD and DevOps workflow.

# Tech Stack:
Node.js, Express, Jest, Supertest, Docker, Jenkins, AWS EC2 (Ubuntu), GitHub

# CI/CD Pipeline:
On each push, Jenkins automatically:

Builds a Docker image
Runs tests inside a container
Deploys to AWS EC2 via SSH

# Highlights:

Real-time Markdown rendering
Automated testing and deployment
Optimized for low-resource instances (swap memory, secure SSH)
